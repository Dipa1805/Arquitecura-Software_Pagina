Introducción

Maven es una herramienta importante para la gestión de proyectos de desarrollo de software con Java. Su principal función en este sentido es la de simplificar y agilizar los procesos en este lenguaje, aunque también es compatible con otros sistemas de programación. Así, Maven da a los programadores una estructura coherente para los proyectos y se encarga de gestionar las dependencias entre distintas tareas.

Objetivos

  • Reificación de Proyecto / Artefacto en forma standard, declarativa y extensible
  • La dependencia de Gson es usada para trabajar con JSON en Java.
  • Las dependencias especifican el groupId, el artifactId y la version de la biblioteca.

Contenidos

  • La dependencia de JUnit es usada para escribir y ejecutar pruebas.
  • Lenguajes de programación para aplicaciones móviles web (HTML, CSS, JavaScript)
  • Frameworks para aplicaciones móviles web (React Native, Ionic, etc.)
  • Diseño de interfaces de usuario para aplicaciones móviles web
  • Desarrollo de aplicaciones móviles web utilizando APIs y bases de datos
  • Pruebas y depuración de aplicaciones móviles web
  • Publicación y distribución de aplicaciones móviles web

Evaluación

La evaluación de esta asignatura se realizará a través de:

  • Exámenes parciales y finales
  • Proyectos individuales
  • Realizacion de ejercicios por semanas
